Starting Out With Python was the book used for the class.

Chapter 1 - Introduction to Computers and Programming

To sum it up, the basics of how a computer works is discussed. Such as the parts of a computer like a CPU or RAM. The differences between volitile and involitile memory, input and output devices.

Chapter 2 - Input, Processing, and Output

This chapter helps understand the bare bones on writing a program. Learning the steps to designing a program like writing psuedocode which is writing fake code to model a program or using a flow chart to depict steps in a program. Learning how to use the print() to output data.

Chapter 3 - Decision Structures and Boolean Logic

We learn how to write condition statements using if, if-else statements. The use of boolean expressions such as >, <, ==, != and many more.

Chapter 4 - Repetition Structures

The use of For-loops and While-loops are used to run repetitious tasks. For-loops are count-controlled loops and While-loops are condition-controlled loops.

For-loop syntax:

for variables in [val1, val2, ...]:

statements

While-loop syntax:

while (condition):

statements

Chapter 6 - Files and Exceptions

Python has functions and methods to read and write files. A file can be accessed three ways: 'r' read, 'w' write, and 'a' append. Reading mode accesses the file in read mode meaning no changes will be made. Write mode gives the option to write data to a file but it will overwrite the file after each save. Append mode safely writes data to an existing file without overwriting the file.

Opening a file:
file_object = open(file_name, mode)

Chapter 7 - Lists and Tuples

Lists (Or sequences) are objects that contain multiple items. Tuples are like lists except they are immutable. You can add more items to a list but not to a tuple.

Creating a list:
list = [item1, item2, ...]

Creating a tuple:
tuple = (item1, item2, ...)

There are methods built in the language to create and alter lists such as:

append(item)# Adds item to list.
index(item) # Returns index num of item if it exists.
remove(item)# Remove item from the list if it exists.

Chater 8 - More about Strings

Strings are also considered sequences. The methods and functions that work with sequences will also work with strings.

Use a for-loop to access an individual character in a string:

for character in string:

string[character]

Character is the index of string and will iterate until there are no more characters.

Chater 9 - Dictionaries and Sets

Dictionaries are objects that store a collection data and are unsorted. Each element has a key and a value. Dictionaries are mutable. Sets are also object and store a collection of data as a mathematical set. All items are unique and can be any data type.

Creating a dictionary:

dictionary = {key1:value1, key2:value2, ...}

Creating a set using the set function

set()
or
set(argument)

The set function can have no argument for an empty set or have an object passed like a tuple or list.

Chater 10 - Classes and Object-Oriented Programming

Object-oriented programming focuses on using objects that contain data attributes and methods that can be interacted by any outside code. A class is code that specifies the data attributes and methods of a particular type in an object. Think of the class as the blueprints to a house and the objects are the houses built using that blueprint.

Set up the class with an initalizer:

class Class_Name:

def __init__ (self):

The initializer method is the first to be written in the class and is automatically executed when an instance of the class is executed. The initializer has a self parameter assigned.

Chater 11 - Inheritance

When programming objects in the real world, many objects are a specialized version of a more general object. Like say a rectangle is a shape or a bee is an insect. This is inheritance, which are classes that share an "is a" relationship.

How Inheritance works:

class Main_Class:

statements

class Sub_Class(Main_Class):

statements

Sub_Class inherits all the general data from Main_Class while adding on its own specialized data. A rectangle would be the specialized class to the general class Shapes.
